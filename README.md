# suprnation-triangle

## Running 


1. Compile the code

``` javac MinTrianglePath.java ```


2. Run the code

``` cat data_big.txt | java MinTrianglePath ```

3. Expected output:
``` 
Minimal path is: 1 + 1 + 1 + 1 +...+ 1 + 1 + 1 = 2000
Reading input time: 0.245 seconds
Processing triangle time: 0.102 seconds 
```
